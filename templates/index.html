Tentang Aplikasi

Aplikasi ini mengimplementasikan sistem cerdas laboratorium Speech-to-Text (STT) dan Text-to-Speech (TTS) mandiri berbasis ekstraksi ciri frekuensi menggunakan:

• MFCC (Mel-Frequency Cepstral Coefficients) untuk menangkap karakteristik unik sinyal vokal ucapan
• DTW (Dynamic Time Warping) dengan restriksi Sakoe-Chiba untuk penyelarasan sumbu waktu pola ucapan
• Dataset Vokal Mandiri sebagai basis data acuan klasifikasi *Nearest Neighbor*

Prosedur Operasional Pengujian
1

🚀 Inisialisasi Environment Server

Pastikan dependensi Python sudah terinstal di komputer lokal, lalu eksekusi berkas backend utama:
pip install flask librosa numpy scipy soundfile
python app.py
Akses antarmuka web melalui: http://127.0.0.1:5000

2

🎙 Akuisisi Korpus Data Latih (Tab "Rekam Suara")

Pilih salah satu token kosakata dari grid menu, tekan tombol lingkaran mikrofon, ucapkan kata target secara natural dan lugas, lalu hentikan perekaman.
Disarankan merekam **minimal 3× hingga 5× per kata** untuk memperkaya model variansi pencocokan DTW.
Total korpus wajib (47 kata): nol, satu, dua, tiga, empat, lima, enam, tujuh, delapan, sembilan, sepuluh, tambah, kurang, kali, bagi, sama-dengan, sayang, semuanya, cleosa, zelda, avrillya, dan abjad A-Z.

3

👂 Evaluasi Speech-to-Text Tersegmentasi (Tab "STT")

Buka panel STT, picu mikrofon, dan ucapkan angka tunggal maupun ekspresi matematika majemuk (contoh: satu tambah nol). Sistem akan memecah sinyal kontinu berdasarkan parameter hening, mengekstrak 39 dimensi koefisien MFCC, dan mengukur jarak penalti DTW terkecil.

4

🔊 Sintesis Akustik Text-to-Speech (Tab "TTS")

Masukkan string teks perintah bebas atau manfaatkan fitur makro instan seperti **NRP** atau **Nama**. Sistem akan menyusun dan memainkan urutan potongan audio WAV secara runut dan mulus.
